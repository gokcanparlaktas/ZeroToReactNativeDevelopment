This repository contains multiple mobile applications I built while following the **"React Native - The Practical Guide" by Maximilian Schwarzmüller** course.

The goal is to practice and apply key concepts such as mobile UI development, navigation, state management, and API integration.

I will continue to update this repository as I progress through the course.

## 🛠️ Tech Stack

- React Native (Expo)
- JavaScript / TypeScript

## 🚀 Getting Started

```bash
npm install
npx expo start
```

### What I learned

- Styling with `StyleSheet`
- Using core React Native components like `View`, `Text`, `TextInput`, `Button`, `Pressable`,`ScrollableView` and `FlatList`
- Managing state with `useState`
- Rendering and updating dynamic lists
- Splitting the app into reusable components
- Passing data and functions with props
- Using `Modal` for input handling

### Project Practice

In this module, I first built a simple **to-do app** as part of the course practice.  
After that, I customized it and turned it into a **cafe notes app** where I can save:

- cafe names
- good sides
- bad sides

I also updated the design to give it a more personalized notebook-style look.
